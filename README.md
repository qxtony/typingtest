# Graphical program for measuring the speed of printing.

## Installation

```bash
git clone https://github.com/qxtony/typingtest.git
```

Go to the folder and install the dependencies:

```bash
cd typingtest
poetry install
```

To start the program, enter the following:
```sh
python -m typingtest
```
